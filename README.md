# rime


##### 配置文件路径:
```
Windows
    Weasel: %APPDATA%\Rime
Mac OS X
    Squirrel: ~/Library/Rime
Linux
    iBus: ~/.config/ibus/rime
    Fcitx: ~/.config/fcitx/rime
```
